# 🚁 Drone Product Landing Page

A modern **product landing page** designed to showcase a professional drone with detailed specifications, comparison tables, customer reviews, and accessories.
The page focuses on delivering a clean UI and responsive layout for an engaging product presentation.

---

## 📌 Overview

This project is a **single product landing page** for a professional drone called **DJI Mavic 4 Pro**, highlighting its key features such as an **8K camera, 40-minute flight time, and AI obstacle avoidance** to attract photographers, filmmakers, and content creators.

The goal of this project is to practice **modern frontend layout techniques, responsive design, and product-focused UI/UX**.

---

## ✨ Features

* Hero section with product image and pricing
* Product highlights and specifications
* Feature comparison table with competitors
* Customer reviews section
* Accessories section with add-to-cart buttons
* Responsive layout for different screen sizes
* Clean and modern UI design

---

## 🖥️ Technologies Used
